# Calculator App (Android)

## Overview
This is a basic calculator Android application built using Java and XML. It performs standard arithmetic operations like addition, subtraction, multiplication, and division.

## Features
- Simple and clean user interface
- Supports basic arithmetic operations
- Clear button to reset calculations
- Handles division by zero

## Technologies Used
- Java
- XML
- Android Studio

## Project Structure
- CalculatorActivity.java → Handles all calculation logic
- activity_calculator.xml → UI layout for calculator

## How It Works
- User inputs numbers using buttons
- Selects an operator
- Presses '=' to get result
- 'C' button resets everything
